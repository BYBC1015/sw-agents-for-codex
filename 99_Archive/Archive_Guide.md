# Archive Guide 归档规范

## 命名规则

```text
项目名_类型_平台_尺寸_版本_日期
```

示例：

```text
新品发布_海报_小红书_3x4_v1.0_2026-05-11.png
新品发布_视频_抖音_9x16_v1.0_2026-05-11.mp4
```

## 版本规则

| 版本 | 含义 |
| --- | --- |
| v0.1 | 内部草稿 |
| v0.5 | 可给内部看 |
| v0.8 | 可给客户/老板看 |
| v1.0 | 已确认交付 |
| v1.1 | 交付后小改 |
| v2.0 | 方向级重做 |

## 归档结构

```text
99_Archive/
  ProjectName/
    source/
    export/
    preview/
    ai_assets/
    references/
    notes/
```
