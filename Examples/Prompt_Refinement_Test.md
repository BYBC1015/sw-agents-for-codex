# 提示词精修测试样例

## 测试目标

验证 Codex 能否把模糊反馈翻译成具体图片/视频参数，而不是直接写“近一点、远一点、高级感”。

## 图片测试

### 原始需求

```text
给我一个新品咖啡海报主视觉提示词，要高级一点，咖啡杯近一点，但不要太花。
```

### 合格输出应包含

- 用途：海报主视觉。
- 景别：近景或中近景。
- 主体占比：例如 60%-70%。
- 构图：咖啡杯偏下或偏侧，保留标题留白。
- 光线：柔和侧逆光或窗边自然光。
- 色彩：低饱和暖棕、奶白、少量深绿或黑。
- 负面提示词：不要文字、不要 logo、不要杂乱背景、不要变形杯子。

### 不合格输出

```text
高级感咖啡海报，咖啡近一点，画面干净，电影感。
```

## 视频测试

### 原始需求

```text
把这个镜头做得更有冲击，镜头靠近产品，适合 5 秒短视频。
```

### 合格输出应包含

- 时长：5 秒。
- 起始画面：产品中景，主体占画面 35%-40%。
- 运镜：快速 push-in，5 秒内主体占比提升到 70%。
- 动作：产品表面高光扫过或液体轻微流动。
- 结束画面：产品近景，logo 区域留给后期叠加，不由 AI 生成。
- 禁止：产品变形、文字错误、跳切、闪烁、多余物体。

### 不合格输出

```text
镜头更近，更炸，更高级，产品很有冲击力。
```
