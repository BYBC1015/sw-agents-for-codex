# Prompt Library 设计自动化提示词库

使用方式：不要一次性把所有提示词都丢给 Codex。按阶段复制对应提示词，并贴入当前文件里的关键内容。

## 0. 低成本启动

```text
你是我的设计项目总控。请先不要生成完整方案，只做低成本判断。

项目原始需求：
[粘贴原话]

请输出：
1. 你理解的项目目标
2. 受众和平台假设
3. 当前缺少的信息
4. 需要我确认的最多 3 个关键问题
5. 是否可以进入创意方向阶段

每项简洁，不要扩写。
```

## 1. Brief 提取

```text
请把以下客户/老板原话整理成可执行 Brief。

原话：
[粘贴原始需求]

请按以下结构输出：
- 真实目标
- 目标受众
- 核心信息
- 必须出现
- 禁止出现
- 情绪关键词
- 风格关键词
- 成功标准
- 待确认问题

要求：
1. 不要脑补无法确定的信息。
2. 对不清楚的信息标记为“待确认”。
3. 输出适合填入 `00_Brief/Brief.md`。
```

## 2. 客户语言翻译成设计语言

```text
请把客户的模糊反馈翻译成设计可执行语言。

客户反馈：
[例如：高级一点、年轻一点、不够炸、感觉太空]

当前方案摘要：
[粘贴方案]

请输出：
1. 这句话可能真正指什么
2. 对版式、色彩、字体、图像、文案的具体影响
3. 需要追问客户的问题
4. 不建议直接执行的误区
```

```text
请输出：

```

## 4. 生成 3 个创意方向

```text
你是资深创意总监。请基于 Brief 和研究结论，生成最多 3 个创意方向。

Brief：
[粘贴]

研究结论：
[粘贴]

请输出：
1. 方向总览表
2. 每个方向的一句话概念
3. 海报呈现
4. 视频呈现
5. 适合原因
6. 风险
7. 推荐排序

限制：
- 不超过 3 个方向。
- 不要进入完整执行细节。
- 明确指出哪个方向最适合先做。
```

## 5. 海报文案生成

```text
请为以下海报方向生成文案方案。

Brief：
[粘贴]

主方向：
[粘贴]

请输出 3 组文案，每组包含：
- 主标题：10 字以内
- 副标题：20 字以内
- 关键信息
- 行动语
- 适合的视觉配合

要求：
文案要有差异，不要只是同义改写。
```

## 7. AI 图像生成提示词

```text
请按 `Prompts/Prompt_Spec_Rules.md` 的规范，把下面的海报视觉方案转成适合 AI 图像生成的提示词。

视觉方案：
[粘贴]

用途：
[海报主视觉 / 背景 / 元素 / 人物 / 产品氛围图]

请输出：
1. 用途判断
2. 中文主提示词
3. 英文主提示词
4. 负面提示词
5. 景别、主体占比、构图、留白说明
6. 可控修改项：更近 / 更远 / 更高级
7. 后期合成注意事项
8. 版权和相似性风险提醒

不要生成品牌 logo、真实名人、受版权保护角色的仿制内容。
不要使用只有“近一点、远一点、高级感、电影感”的模糊描述，必须翻译成具体视觉参数。
```

## 9.1 AI 视频镜头提示词

```text
请按 `Prompts/Prompt_Spec_Rules.md`，把以下分镜转成 AI 视频镜头提示词。

分镜：
[粘贴]

平台/画幅/时长：
[粘贴]

请输出：
1. 每个镜头一条提示词
2. 每条包含起始画面、动作、镜头运动、结束画面
3. 每条包含景别、主体占比、运镜速度、光线、色彩和连续性要求

- 不要把多个复杂镜头塞进一条 prompt。
- 不要只写“镜头靠近、远一点、电影感”。
- 必须转成具体运镜和画面参数。
```
