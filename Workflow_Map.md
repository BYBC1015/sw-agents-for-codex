# 自动化流程地图

## 角色分工

| 阶段 | 你负责 | Codex 负责 | 输出 |
| --- | --- | --- | --- |
| 需求澄清 | 判断真实目标、补充商业背景 | 整理需求、发现缺口、生成追问 | `00_Brief/Brief.md` |
| 风格研究 | 选择审美方向和参考质量 | 汇总参考、提炼风格语言、提示风险 | `01_Research/Research_Board.md` |
| 创意方向 | 决定主方向 | 生成不超过 3 个方向并做取舍建议 | `02_Direction/Creative_Direction.md` |
| 海报设计 | 做最终审美判断和设计执行 | 产出文案、版式、元素、AI 图像提示词 | `03_Poster/Poster_Workflow.md` |
| 视频分镜 | 判断叙事节奏和镜头可执行性 | 生成脚本、镜头表、转场、素材需求 | `04_Storyboard/Storyboard.md` |
| 剪辑方案 | 实际剪辑、调色、包装 | 给出时间线、字幕、音效、节奏建议 | `05_EditPlan/Edit_Plan.md` |
| 终审交付 | 做最终把关 | 检查品牌、版权、平台规格、遗漏项 | `06_FinalReview/Final_Review_Checklist.md` |
| 改稿归档 | 决定是否接受变更 | 评估影响范围、保留版本记录 | `Change_Log.md` |

## 阶段确认点

| 确认点 | 必须确认 | 未确认时禁止 |
| --- | --- | --- |
| G1 Brief 锁定 | 目标、受众、平台、尺寸、截止时间 | 大量生成创意变体 |
| G2 方向锁定 | 主方向、禁用方向、参考风格 | 生成最终图像或完整分镜 |
| G3 方案锁定 | 海报主版式或视频主脚本 | 批量做精修和导出 |
| G4 交付锁定 | 文件规格、版权、品牌、平台规范 | 对外发送最终稿 |

## 改稿处理逻辑

1. 先写入 `Change_Log.md`。
2. 判断属于文案微调、局部视觉调整、结构变化还是方向重做。
3. 估算 token、设计时间和素材影响。
4. 若影响 G2 之前的方向判断，必须重新生成 `Creative_Direction.md` 的相关部分。
5. 若只影响 G3 后的局部执行，只更新对应文件并保留旧版。

## 工具适配

| 工具 | Codex 适合做什么 | 你在工具里做什么 |
| --- | --- | --- |
| Photoshop / Illustrator | 版式建议、图层命名、视觉元素清单、生成式填充提示词 | 精修、合成、字体和品牌控制 |
| Premiere / After Effects | 时间线、镜头节奏、字幕、转场、动效说明 | 剪辑、调色、包装、渲染 |
| Canva | 模板结构、文案变体、尺寸改版清单 | 快速排版、多人协作、批量导出 |
| 剪映 / 短视频工具 | 口播脚本、字幕节奏、BGM 和音效点 | 快速剪辑、平台化包装、发布 |
