# Final Review 终审检查

## 内容检查

- [ ] 核心信息准确。
- [ ] 时间、地点、价格、活动规则准确。
- [ ] 品牌名称、产品名称、人名无误。
- [ ] 文案无错别字。
- [ ] 没有与 Brief 冲突的内容。

## 视觉检查

- [ ] 第一眼重点明确。
- [ ] 信息层级清楚。
- [ ] 字体、颜色、logo 符合品牌限制。
- [ ] 移动端预览可读。
- [ ] 没有明显压缩、拉伸、糊图。

## 视频检查

- [ ] 前 3 秒有钩子。
- [ ] 字幕可读且无错字。
- [ ] 音量平衡。
- [ ] 转场和动效不过度。
- [ ] 节奏符合平台。
- [ ] 结尾有清楚行动或记忆点。

## AI 与版权检查

- [ ] AI 生成素材已标记来源。
- [ ] 人像、品牌、字体、音乐授权已确认。
- [ ] 没有明显侵权参考。
- [ ] 没有平台敏感内容。
- [ ] 没有错误事实或夸大承诺。
